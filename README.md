# mos
Modules Oriented microService Framework   
Power by Spring Boot

# 项目工程清单说明  
* mos-build  
项目群构建控制
* mos-parent  
maven parent project for mos framework project groups.
* mos-models  
mos项目群内部顶组沟通模型
* mos-core  
mos核心框架
* mos-autoconfig  
mos的spring-boot-autoconfig自动配置项目
* mos-starters/???-starter  
spring-boot-starter项目清单  
     * mos-starter  
  mos核心框架starter
* mos-example
mos的示例应用项目
